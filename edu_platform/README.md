# 项目简介
当前项目主要是为edu项目提供能力的后端微服务。[edu](https://github.com/liangfutang/web-project)

# 相关命令
查看当前项目的虚拟环境
```shell
pipenv --venv
```
为项目安装虚拟环境
```shell
pipenv install
```
卸载当前虚拟环境
```shell
pipenv --rm
```

项目创建
```shell
Django-admin startproject django
```
服务启动
```shell
python manage.py runserver 0.0.0.0:8080
```
新建模块
```shell
python manage.py startapp user
```
